# 작업 로그 (Work Log)

**목적**: 세션이 끊어져도 작업 내용을 추적할 수 있도록 모든 주요 작업을 기록

**사용 방법**:
- 모든 Claude Code 세션은 작업 시작 시 이 파일을 읽어야 함
- 작업 완료 시 이 파일에 기록해야 함
- 최신 작업이 맨 위에 오도록 역순 정렬

---
