# 회원 정치인 활동 유저플로우

> 정치인 목록 조회, 상세 조회, 관심 정치인 등록, 평점 매기기

---

## 1. 정치인 목록 조회

```
/politicians 페이지 접속
    ↓
GET /api/politicians → 정치인 목록 로드
    ↓
필터/검색 옵션:
- 정당별 필터
- 지역별 필터
- 직위별 필터
- 이름 검색
    ↓
정치인 카드 목록 표시 → 페이지네이션
```

---

## 2. 정치인 상세 조회

```
정치인 목록에서 클릭 → /politicians/[id]
    ↓
GET /api/politicians/[id] → 상세 정보 로드
    ↓
표시 항목:
- 기본 정보 (이름, 정당, 직위, 지역)
- 프로필 사진
- 경력 사항
- 공약 목록
- 평점/평가
- 관련 게시글
```

---

## 3. 관심 정치인 등록

```
정치인 상세 페이지 → ♥ 관심 버튼 클릭
    ↓
로그인 확인
├─ 비로그인: 로그인 페이지로 이동
└─ 로그인: 계속
    ↓
POST /api/favorites → 관심 등록
    ↓
버튼 상태 변경 (♥ 채워짐) → 완료
```

**취소**: 다시 클릭 → DELETE /api/favorites → 관심 해제

---

## 4. 정치인 평점 매기기

```
정치인 상세 페이지 → 평점 영역
    ↓
로그인 확인
├─ 비로그인: 로그인 유도
└─ 로그인: 계속
    ↓
별점 선택 (1~5점) → POST /api/ratings/[politicianId]
    ↓
평균 평점 업데이트 → 완료
```

**수정**: 다시 별점 선택 → 기존 평점 업데이트

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 목록 조회 | GET | /api/politicians |
| 상세 조회 | GET | /api/politicians/[id] |
| 검색 | GET | /api/search/politicians |
| 관심 등록 | POST | /api/favorites |
| 관심 해제 | DELETE | /api/favorites |
| 관심 목록 | GET | /api/favorites |
| 평점 등록 | POST | /api/ratings/[politicianId] |
| 평점 조회 | GET | /api/ratings/[politicianId] |
