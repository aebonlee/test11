# 회원 인증 유저플로우

> 회원가입, 소셜로그인, 이메일로그인, 비밀번호재설정, 로그아웃

---

## 1. 회원가입

```
비회원 → 회원가입 클릭 → /auth/signup
    ↓
이메일/비밀번호/이름 입력 → 약관 동의 → 가입 버튼
    ↓
POST /api/auth/signup → 인증 이메일 발송
    ↓
이메일 링크 클릭 → 인증 완료 → 로그인 가능
```

**입력 필드**: 이메일, 비밀번호(8자+), 비밀번호확인, 이름

---

## 2. 소셜 로그인

```
로그인 페이지 → 소셜 버튼 클릭 (Google/Kakao/Naver)
    ↓
소셜 서비스 인증 페이지 → 로그인/권한승인
    ↓
/auth/callback → 토큰 검증
    ↓
신규: 자동 회원가입 → 기존: 로그인 처리 → 메인 이동
```

---

## 3. 이메일 로그인

```
/auth/login → 이메일/비밀번호 입력 → 로그인 버튼
    ↓
POST /api/auth/login → 인증 확인
    ├─ 실패: 오류 메시지
    └─ 성공: 세션 생성 → 메인 이동
```

---

## 4. 비밀번호 재설정

```
로그인 페이지 → "비밀번호 찾기" 클릭 → /auth/password-reset
    ↓
이메일 입력 → POST /api/auth/password-reset → 재설정 이메일 발송
    ↓
이메일 링크 클릭 → /auth/password-reset/form
    ↓
새 비밀번호 입력 → POST /api/auth/password-reset/confirm → 완료
```

---

## 5. 로그아웃

```
프로필 메뉴 → 로그아웃 클릭
    ↓
POST /api/auth/logout → 세션 삭제 → 메인 이동
```

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 회원가입 | POST | /api/auth/signup |
| 로그인 | POST | /api/auth/login |
| 로그아웃 | POST | /api/auth/logout |
| 비밀번호 재설정 요청 | POST | /api/auth/password-reset |
| 비밀번호 재설정 확인 | POST | /api/auth/password-reset/confirm |
| 소셜 콜백 | GET | /auth/callback |
