# 회원 소셜/기타 유저플로우

> 팔로우, 공유, 알림 확인, 문의하기

---

## 1. 팔로우하기

```
다른 사용자 프로필 페이지 → "팔로우" 버튼 클릭
    ↓
로그인 확인 (필수)
    ↓
POST /api/users/[id]/follow
    ↓
버튼 상태 변경 ("팔로잉") → 완료
```

**언팔로우**: "팔로잉" 버튼 클릭 → DELETE /api/users/[id]/follow

---

## 2. 공유하기

```
게시글/정치인 상세 → 공유 버튼 클릭
    ↓
공유 옵션 모달:
- 링크 복사
- 카카오톡 공유
- 트위터 공유
- 페이스북 공유
    ↓
선택한 방식으로 공유 → POST /api/shares (통계 기록)
    ↓
완료 메시지 표시
```

---

## 3. 알림 확인

```
헤더 알림 아이콘 클릭 → /notifications
    ↓
GET /api/notifications → 알림 목록 로드
    ↓
알림 종류:
- 댓글 알림 (내 글에 댓글)
- 답글 알림 (내 댓글에 답글)
- 팔로우 알림 (새 팔로워)
- 공감 알림 (내 글에 공감)
    ↓
알림 클릭 → 해당 페이지로 이동 + 읽음 처리
```

**읽음 처리**: PATCH /api/notifications/[id]
**전체 읽음**: PATCH /api/notifications/read-all

---

## 4. 문의하기

```
고객지원 메뉴 → /support
    ↓
문의 유형 선택:
- 서비스 이용 문의
- 오류 신고
- 건의사항
- 기타
    ↓
문의 내용 작성:
- 제목
- 내용
- 연락처 (선택)
- 첨부파일 (선택)
    ↓
"문의하기" 클릭 → POST /api/inquiries
    ↓
접수 완료 메시지 → 이메일로 답변 안내
```

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 팔로우 | POST | /api/users/[id]/follow |
| 언팔로우 | DELETE | /api/users/[id]/follow |
| 팔로워 목록 | GET | /api/users/[id]/followers |
| 팔로잉 목록 | GET | /api/users/[id]/following |
| 공유 기록 | POST | /api/shares |
| 알림 목록 | GET | /api/notifications |
| 알림 읽음 | PATCH | /api/notifications/[id] |
| 전체 읽음 | PATCH | /api/notifications/read-all |
| 문의 등록 | POST | /api/inquiries |
