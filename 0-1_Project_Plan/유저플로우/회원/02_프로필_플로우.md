# 회원 프로필 유저플로우

> 프로필 조회, 프로필 수정, 회원 탈퇴

---

## 1. 프로필 조회

```
로그인 상태 → 마이페이지 클릭 → /mypage
    ↓
GET /api/user/profile → 프로필 정보 로드
    ↓
표시 항목:
- 프로필 이미지, 이름, 이메일, 가입일
- 활동 통계 (게시글/댓글/관심정치인)
- 팔로워/팔로잉 수
```

---

## 2. 프로필 수정

```
마이페이지 → "프로필 수정" 클릭 → /profile/edit
    ↓
현재 정보 로드 → 수정 입력
- 프로필 이미지 변경
- 이름 변경
- 소개 변경
    ↓
저장 클릭 → PATCH /api/user/profile/update → 완료
```

**이미지 업로드**: POST /api/upload/profile (5MB 이하)

---

## 3. 회원 탈퇴

```
설정 페이지 → "회원 탈퇴" 클릭 → /settings
    ↓
비밀번호 확인 모달 → 비밀번호 입력
    ↓
"탈퇴하기" 클릭 → DELETE /api/user/delete
    ↓
계정 삭제 완료 → 로그아웃 → 메인 이동
```

**주의**: 탈퇴 시 모든 데이터 삭제 (복구 불가)

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 프로필 조회 | GET | /api/user/profile |
| 프로필 수정 | PATCH | /api/user/profile/update |
| 이미지 업로드 | POST | /api/upload/profile |
| 활동 통계 | GET | /api/users/[id]/stats |
| 회원 탈퇴 | DELETE | /api/user/delete |
