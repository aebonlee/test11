# 정치인 인증 유저플로우

> 정치인 본인 인증, 인증 코드 확인, 인증 상태 조회

---

## 1. 정치인 본인 인증 요청

```
정치인 상세 페이지 → "본인 인증하기" 버튼
    ↓
인증 정보 입력:
- 이름 (정치인 DB와 일치 확인)
- 소속 정당
- 출마 직종/직위
    ↓
POST /api/politicians/verify-simple
    ↓
인증 요청 접수 → 관리자 승인 대기
```

---

## 2. 이메일 인증 코드 방식

```
정치인 인증 페이지 → 공식 이메일 입력
    ↓
POST /api/politicians/verify/send-code → 인증 코드 발송
    ↓
이메일에서 6자리 코드 확인
    ↓
코드 입력 → POST /api/politicians/verify/check-code
    ├─ 실패: "코드가 일치하지 않습니다"
    └─ 성공: 인증 완료
```

---

## 3. 인증 상태 조회

```
정치인 프로필 페이지
    ↓
GET /api/politicians/verify-session
    ↓
인증 상태 표시:
- 미인증: "인증하기" 버튼 표시
- 대기중: "승인 대기중" 배지
- 승인됨: ✓ 인증됨 배지
- 거부됨: "인증 거부됨" + 재신청 버튼
```

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 간편 인증 | POST | /api/politicians/verify-simple |
| 코드 발송 | POST | /api/politicians/verify/send-code |
| 코드 확인 | POST | /api/politicians/verify/check-code |
| 세션 확인 | GET | /api/politicians/verify-session |
