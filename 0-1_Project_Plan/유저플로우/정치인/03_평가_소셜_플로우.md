# 정치인 평가/소셜 유저플로우

> 평점 확인, AI 평가, 댓글 관리, 관심/팔로워 확인

---

## 1. 평점 확인

```
정치인 상세 페이지 → 평점 영역
    ↓
GET /api/ratings/[politicianId]
    ↓
표시 정보:
- 평균 평점 (5점 만점)
- 총 평가 수
- 평점 분포 (별점별 비율)
```

---

## 2. AI 평가 조회

```
정치인 상세 페이지 → "AI 분석" 탭
    ↓
GET /api/politicians/[id]/ai-evaluation
    ↓
표시 정보:
- 종합 평가 점수
- 항목별 점수 (공약 이행, 의정 활동 등)
- AI 코멘트
- 분석 기준일
```

---

## 3. 댓글 관리 (인증된 정치인)

```
본인 프로필의 댓글 목록
    ↓
GET /api/politicians/[id]/comments
    ↓
가능한 작업:
- 댓글에 답글 달기
- 부적절한 댓글 신고
```

---

## 4. 관심/팔로워 확인

```
정치인 프로필 페이지
    ↓
관심 등록 수: GET /api/politicians/[id]/favorites/count
    ↓
표시:
- "♥ 1,234명이 관심"
- 최근 관심 등록한 사용자 프로필
```

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 평점 조회 | GET | /api/ratings/[politicianId] |
| AI 평가 | GET | /api/politicians/[id]/ai-evaluation |
| 댓글 목록 | GET | /api/politicians/[id]/comments |
| 관심 수 | GET | /api/politicians/[id]/favorites/count |
| 통계 | GET | /api/statistics/politicians |
