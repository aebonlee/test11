# 정치인 프로필 유저플로우

> 프로필 조회, 프로필 수정, 경력 관리, 공약 관리

---

## 1. 프로필 조회 (일반 사용자)

```
정치인 목록에서 선택 → /politicians/[id]
    ↓
GET /api/politicians/[id] → 프로필 로드
    ↓
표시 정보:
- 기본 정보 (이름, 정당, 직위)
- 프로필 사진
- 연락처/SNS
- 경력 사항
- 공약 목록
- 평점/리뷰
```

---

## 2. 프로필 수정 (인증된 정치인)

```
본인 프로필 페이지 → "수정" 클릭 → /politicians/[id]/edit
    ↓
인증 상태 확인 (인증된 정치인만 가능)
    ↓
수정 가능 항목:
- 프로필 사진
- 연락처
- SNS 링크
- 소개글
    ↓
저장 → PATCH /api/politicians/[id]/profile → 완료
```

---

## 3. 경력 관리

```
프로필 수정 페이지 → "경력" 탭
    ↓
경력 추가:
- 기간 (시작~종료)
- 직위/직책
- 소속 기관
- 설명
    ↓
POST /api/politicians/[id]/careers
    ↓
경력 목록 업데이트 → 완료
```

**수정**: PATCH /api/politicians/[id]/careers/[careerId]
**삭제**: DELETE /api/politicians/[id]/careers/[careerId]

---

## 4. 공약 관리

```
프로필 수정 페이지 → "공약" 탭
    ↓
공약 추가:
- 공약 제목
- 분야 (경제, 복지, 교육 등)
- 상세 내용
- 이행 상태 (예정/진행중/완료)
    ↓
POST /api/politicians/[id]/pledges
    ↓
공약 목록 업데이트 → 완료
```

---

## API 목록

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
| 프로필 조회 | GET | /api/politicians/[id] |
| 프로필 수정 | PATCH | /api/politicians/[id]/profile |
| 경력 추가 | POST | /api/politicians/[id]/careers |
| 경력 수정 | PATCH | /api/politicians/[id]/careers/[id] |
| 공약 추가 | POST | /api/politicians/[id]/pledges |
| 공약 수정 | PATCH | /api/politicians/[id]/pledges/[id] |
