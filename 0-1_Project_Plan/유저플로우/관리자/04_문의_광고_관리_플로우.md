# 관리자 문의/광고 관리 유저플로우

> 문의 관리, 광고 관리

---

## 1. 문의 목록 조회

```
/admin/inquiries 페이지 접속
    ↓
GET /api/admin/inquiries → 문의 목록 로드
    ↓
필터: 상태별 (대기/처리중/완료), 유형별
    ↓
문의 목록 표시 (최신순)
```

---

## 2. 문의 상세/답변

```
문의 목록에서 선택 → 상세 보기
    ↓
GET /api/admin/inquiries/[id]
    ↓
표시 정보:
- 문의자 정보
- 문의 유형
- 문의 내용
- 첨부파일
    ↓
답변 작성 → "답변 등록" 클릭
    ↓
PATCH /api/admin/inquiries/[id]
    ↓
상태 변경 (완료) + 이메일 발송
```

---

## 3. 광고 목록 조회

```
관리자 → 광고 관리
    ↓
GET /api/admin/ads → 광고 목록 로드
    ↓
필터: 상태별 (활성/비활성/종료), 위치별
    ↓
광고 목록 표시
```

---

## 4. 광고 등록/수정

```
"새 광고 등록" 클릭 → 등록 폼
    ↓
입력 항목:
- 광고명
- 광고 이미지
- 링크 URL
- 노출 위치 (메인/사이드바 등)
- 노출 기간
- 상태 (활성/비활성)
    ↓
POST /api/admin/ads → 등록 완료
```

**수정**: PATCH /api/admin/ads/[id]
**삭제**: DELETE /api/admin/ads/[id]

---

## 5. 광고 통계

```
광고 관리 → 통계 탭
    ↓
GET /api/admin/ads/stats
    ↓
표시 항목:
- 노출 수
- 클릭 수
- CTR (클릭률)
- 기간별 추이 차트
```

---
