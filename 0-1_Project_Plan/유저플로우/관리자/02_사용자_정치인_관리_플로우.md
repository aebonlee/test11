# 관리자 사용자/정치인 관리 유저플로우

> 사용자 관리, 정치인 관리, 인증 승인

---

## 1. 사용자 목록 조회

```
/admin/users 페이지 접속
    ↓
GET /api/admin/users → 사용자 목록 로드
    ↓
필터 옵션:
- 상태별 (활성/정지/탈퇴)
- 역할별 (일반/관리자)
- 검색 (이름/이메일)
    ↓
페이지네이션 → 사용자 목록 표시
```

---

## 2. 사용자 상세/수정

```
사용자 목록에서 선택 → 상세 보기
    ↓
GET /api/admin/users/[id]
    ↓
표시 정보:
- 기본 정보 (이름, 이메일, 가입일)
- 활동 내역 (게시글, 댓글)
- 계정 상태
    ↓
수정 가능 항목:
- 역할 변경 (일반 ↔ 관리자)
- 계정 상태 변경 (활성/정지)
    ↓
PATCH /api/admin/users/[id] → 저장
```

---

## 3. 사용자 정지/해제

```
사용자 상세 → "계정 정지" 클릭
    ↓
정지 사유 입력 (선택)
    ↓
PATCH /api/admin/users/[id] { status: 'suspended' }
    ↓
정지 완료 → 해당 사용자 로그인 불가
```

**해제**: status: 'active'로 변경

---

## 4. 정치인 목록 조회

```
/admin/politicians 페이지 접속
    ↓
GET /api/admin/politicians → 정치인 목록 로드
    ↓
필터: 정당별, 직위별, 인증상태별
    ↓
정치인 목록 표시
```

---

## 5. 정치인 등록/수정

```
"새 정치인 추가" 클릭 → 등록 폼
    ↓
입력 항목:
- 이름, 정당, 직위, 지역
- 연락처, 이메일
- 프로필 사진
    ↓
POST /api/admin/politicians → 등록 완료
```

**수정**: PATCH /api/admin/politicians/[id]
**삭제**: DELETE /api/admin/politicians/[id]

---

## 6. 정치인 인증 승인

```
/admin → 인증 요청 목록
    ↓
GET /api/admin/verify-email → 대기중 요청 목록
    ↓
요청 상세 확인:
- 신청자 정보
- 인증 방식
- 제출 자료
    ↓
"승인" 또는 "거부" 클릭
    ↓
PATCH /api/admin/verify-email → 상태 변경
```
