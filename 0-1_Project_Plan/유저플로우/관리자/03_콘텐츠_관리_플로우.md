# 관리자 콘텐츠 관리 유저플로우

> 게시글 관리, 댓글 관리, 자동 검열, 공지사항 관리

---

## 1. 게시글 관리

```
/admin/posts 페이지 접속
    ↓
GET /api/admin/posts → 게시글 목록 로드
    ↓
필터: 카테고리별, 상태별, 신고된 글
    ↓
게시글 선택 → 상세 보기
    ↓
가능한 작업:
- 내용 수정: PATCH /api/admin/posts/[id]
- 삭제: DELETE /api/admin/posts/[id]
- 숨김 처리: status 변경
```

---

## 2. 댓글 관리

```
/admin → 댓글 관리 메뉴
    ↓
GET /api/admin/comments → 댓글 목록 로드
    ↓
필터: 신고된 댓글, 최근 댓글
    ↓
댓글 선택 → 상세 보기
    ↓
가능한 작업:
- 삭제: DELETE /api/admin/comments/[id]
- 신고 처리: 경고/삭제
```

---

## 3. 자동 검열 설정

```
관리자 설정 → 자동 검열
    ↓
GET /api/admin/auto-moderate → 현재 설정 로드
    ↓
설정 항목:
- 금지어 목록 관리 (추가/삭제)
- 자동 숨김 활성화
- 신고 임계값 설정
    ↓
PATCH /api/admin/auto-moderate → 설정 저장
```

---

## 4. 공지사항 관리

```
관리자 → 공지사항 관리
    ↓
GET /api/notices → 공지사항 목록
    ↓
"새 공지 작성" 클릭:
- 제목 입력
- 내용 입력 (에디터)
- 중요 공지 여부
- 노출 기간 설정
    ↓
POST /api/notices → 공지 등록
    ↓
/notices 페이지에 노출
```

**수정**: PATCH /api/notices/[id]
**삭제**: DELETE /api/notices/[id]
