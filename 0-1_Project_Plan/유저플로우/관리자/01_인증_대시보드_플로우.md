# 관리자 인증/대시보드 유저플로우

> 관리자 로그인, 로그아웃, 대시보드, 통계 조회

---

## 1. 관리자 로그인

```
/admin/login 페이지 접속
    ↓
관리자 계정 입력 (이메일/비밀번호)
    ↓
로그인 버튼 클릭 → 권한 확인
    ├─ 일반 회원: "관리자 권한이 없습니다"
    └─ 관리자: 대시보드로 이동
    ↓
/admin (대시보드) 페이지 이동
```

---

## 2. 관리자 로그아웃

```
관리자 메뉴 → 로그아웃 클릭
    ↓
세션 종료 → /admin/login 페이지로 이동
```

---

## 3. 대시보드 조회

```
/admin 페이지 접속
    ↓
GET /api/admin/dashboard → 통계 데이터 로드
    ↓
표시 항목:
- 전체 회원 수 / 신규 가입
- 전체 정치인 수
- 전체 게시글 수 / 오늘 작성
- 전체 댓글 수
- 미처리 문의 수
- 최근 가입자 목록
- 최근 게시글 목록
```

---

## 4. 실시간 통계

```
대시보드 → 통계 차트 영역
    ↓
기간 선택: 일별 / 주별 / 월별
    ↓
차트 표시:
- 가입자 추이
- 게시글 작성 추이
- 방문자 추이
```

---

| 기능 | 메서드 | 엔드포인트 |
|------|--------|------------|
