## 📦 기대 결과물

**생성할 파일:**
1. `playwright.config.ts` - Playwright 설정
2. `e2e/auth.spec.ts` - 인증 시나리오
3. `e2e/politicians.spec.ts` - 정치인 관련 시나리오
4. `e2e/posts.spec.ts` - 게시글 관련 시나리오
5. `e2e/admin.spec.ts` - 관리자 시나리오

**설치 필요 패키지:**
```bash
npm install -D @playwright/test
npx playwright install
```

**package.json 스크립트:**
```json
{
  "scripts": {
    "test:e2e": "playwright test",
    "test:e2e:ui": "playwright test --ui",
    "test:e2e:debug": "playwright test --debug"
  }
}
```

**테스트 시나리오:**
1. **auth.spec.ts**: 회원가입, 로그인, 로그아웃, 비밀번호 찾기
2. **politicians.spec.ts**: 정치인 검색, 필터링, 상세 조회, 즐겨찾기
3. **posts.spec.ts**: 게시글 작성, 수정, 삭제, 댓글, 공감
4. **admin.spec.ts**: 신고 처리, 콘텐츠 삭제, 사용자 정지

**구현해야 할 세부 항목**:

1. Playwright 설정 (playwright.config.ts, 브라우저 설정)
2. 회원가입/로그인 시나리오 (auth.spec.ts)
3. 정치인 검색/상세 조회 시나리오 (politicians.spec.ts)
4. 게시글 작성/댓글 시나리오 (posts.spec.ts)
5. 관리자 대시보드 시나리오 (admin.spec.ts)
6. CI/CD 통합 (.github/workflows/e2e.yml)

각 항목을 체계적으로 구현하고 테스트하세요.
