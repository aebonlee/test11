## 작업 개요
팔로우 시스템의 프론트엔드 UI 및 실시간 기능 구현

## 구현 범위

### 완료된 항목 ✅
1. **FollowButton 컴포넌트**
   - 팔로우/언팔로우 토글
   - 호버 시 "언팔로우" 표시
   - 로딩 상태

2. **팔로워 목록 페이지** (`/users/[id]/followers`)
   - API 연동
   - 페이지네이션
   - 각 팔로워별 FollowButton

3. **팔로잉 목록 페이지** (`/users/[id]/following`)
   - 필터 탭 (전체/회원/정치인)
   - API 연동
   - 페이지네이션

4. **마이페이지 연동**
   - 팔로워/팔로잉 수 클릭 시 목록 페이지로 이동
   - 실제 통계 데이터 표시

### 남은 항목 ⏳
1. **등급 승급 모달**
   - 등급 변동 시 축하 팝업

2. **실시간 알림**
   - 팔로워 획득 토스트
   - 포인트 획득 애니메이션

3. **WebSocket 연동** (선택)
   - 실시간 팔로워 수 업데이트
