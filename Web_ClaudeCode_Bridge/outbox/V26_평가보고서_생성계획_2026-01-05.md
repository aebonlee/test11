# V26.0 AI 평가보고서 생성 계획

**작성일**: 2026-01-05
**버전**: V26.0 (풀링 방식)

---

## 1. 개요

V26.0 풀링 방식으로 수집/평가된 데이터를 기반으로 **AI별 개별 평가보고서**를 생성하는 계획입니다.

### 핵심 원칙
- **AI별 개별 보고서**: 4개 AI (Claude, ChatGPT, Grok, Gemini) 각각 별도 보고서 생성
- **데이터 기반 강점/약점 분석**: 카테고리별 점수에서 자동 도출
- **근거 데이터 포함**: 모든 평가에 대한 출처와 평가 이유 명시

## 5. 강점/약점 분석 방식

### 5.1 자동 도출 로직
```python
# 카테고리별 평균 점수 계산
cat_scores = {카테고리: 평균점수}

# 정렬
sorted_cats = sorted(cat_scores.items(), key=lambda x: x[1], reverse=True)

# 강점: 상위 3개
strengths = sorted_cats[:3]

# 약점: 하위 3개
weaknesses = sorted_cats[-3:]
```

### 5.2 평가 기준
| 점수 범위 | 평가 |
|---|---|
| 85점 이상 | 매우 우수 |
| 80~84점 | 우수 |
| 75~79점 | 양호 |
| 70~74점 | 보통 |
| 70점 미만 | 미흡 |
