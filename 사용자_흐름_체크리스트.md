## 🔍 흐름 1: 정치인 조회 및 평가

```
[홈] → [정치인 목록] → [정치인 상세] → [평가/즐겨찾기]
  │         │              │              │
  │         │              │              └─ 별점 평가 (1-5점)
  │         │              │              └─ 즐겨찾기 추가/삭제
  │         │              │
  │         │              └─ AI 평가 점수 (Claude/ChatGPT/Grok)
  │         │              └─ 회원 평점
  │         │              └─ 경력/학력/공약
  │         │              └─ 커뮤니티 활동
  │         │
  │         └─ 필터 (정당/지역/등급)
  │         └─ 정렬 (점수순/이름순/평점순)
  │         └─ 검색
  │
  └─ TOP 10 정치인 표시
```

### ✅ 체크 항목:

#### 1.1 홈 → 정치인 목록
- [ ] TOP 10 정치인 클릭 시 `/politicians` 이동
- [ ] "더보기" 버튼 클릭 시 `/politicians` 이동
- [ ] 하드코딩된 데이터 없이 API로 로드
- [ ] 로딩 상태 표시
- [ ] 에러 처리

#### 1.2 정치인 목록 페이지
- [ ] 30명 실제 정치인 표시
- [ ] AI 점수 (Claude/ChatGPT/Grok) 정상 표시
- [ ] 회원 평점 정상 표시
- [ ] 필터 동작 (정당/지역/등급)
- [ ] 정렬 동작 (점수순/이름순/평점순)
- [ ] 검색 기능
- [ ] 페이지네이션

#### 1.3 정치인 상세 페이지
- [ ] 정치인 기본 정보 (이름/정당/지역/직책)
- [ ] AI 평가 점수 (하드코딩 제거됨?)
- [ ] 회원 평점 (하드코딩 제거됨?)
- [ ] 경력/학력 표시
- [ ] 공약 표시
- [ ] 커뮤니티 활동 통계

#### 1.4 평가 기능
- [ ] 별점 평가 (1-5점) 동작
- [ ] 로그인 필요 체크
- [ ] 평가 후 평균 점수 업데이트
- [ ] 중복 평가 방지 또는 재평가

#### 1.5 즐겨찾기 기능
- [ ] 즐겨찾기 추가/삭제 동작
- [ ] 로그인 필요 체크
- [ ] 마이페이지에서 확인 가능

## 💬 흐름 2: 커뮤니티 (게시글/댓글)

```
[홈] → [커뮤니티] → [게시글 목록] → [게시글 상세] → [댓글]
  │         │            │               │              │
  │         │            │               │              └─ 댓글 작성
  │         │            │               │              └─ 댓글 수정/삭제
  │         │            │               │
  │         │            │               └─ 게시글 내용
  │         │            │               └─ 공감/싫어요
  │         │            │               └─ 조회수
  │         │            │
  │         │            └─ 필터 (전체/정치인/일반)
  │         │            └─ 정렬 (최신순/공감순/조회순)
  │         │            └─ 검색
  │         │
  │         └─ 글쓰기 → [정치인 게시판/회원 자유게시판 선택]
  │
  └─ 최신 게시글 5개 (정치인 3개 + 인기글 2개)
```

### ✅ 체크 항목:

#### 2.1 홈 → 커뮤니티
- [ ] 최신 게시글 클릭 시 상세 페이지 이동
- [ ] "더보기" 버튼 클릭 시 `/community` 이동
- [ ] 정치인 게시글/일반 게시글 구분 표시

#### 2.2 커뮤니티 목록
- [ ] 전체/정치인/일반 탭 동작
- [ ] 게시글 목록 표시 (프로토타입 스타일)
- [ ] 정렬 기능 (최신순/공감순/조회순)
- [ ] 검색 기능
- [ ] Hot/Best 배지 표시
- [ ] 작성자 표시 (가상 정치인 3명 + 일반 회원 3명)

#### 2.3 글쓰기
- [ ] "글쓰기" 버튼 클릭 시 카테고리 선택 모달
- [ ] 정치인 게시판 선택 시 `/community/posts/create-politician`
- [ ] 회원 자유게시판 선택 시 `/community/posts/create`
- [ ] 로그인 필요 체크

#### 2.4 게시글 상세
- [ ] 제목/내용 표시
- [ ] 작성자 정보 (정치인 또는 일반 회원)
- [ ] 공감/싫어요 동작
- [ ] 조회수 증가
- [ ] 공유 기능

#### 2.5 댓글
- [ ] 댓글 목록 표시
- [ ] 댓글 작성 (일반 회원만, 로그인 필요)
- [ ] 댓글 수정/삭제 (본인만)
- [ ] 대댓글 기능

## 👤 흐름 3: 회원 관리

```
[회원가입] → [이메일 인증] → [로그인] → [마이페이지]
     │            │              │           │
     │            │              │           ├─ 내 정보
     │            │              │           ├─ 내 게시글
     │            │              │           ├─ 내 댓글
     │            │              │           ├─ 즐겨찾는 정치인
     │            │              │           ├─ 내 평가
     │            │              │           └─ 설정
     │            │              │
     │            │              └─ Google/일반 로그인
     │            │
     │            └─ 이메일 인증 링크 클릭
     │
     └─ 이메일/비밀번호/닉네임
```

### ✅ 체크 항목:

#### 3.1 회원가입
- [ ] 이메일/비밀번호/닉네임 입력
- [ ] 유효성 검사
- [ ] 중복 이메일 체크
- [ ] 약관 동의 체크
- [ ] 회원가입 성공 시 이메일 인증 안내

#### 3.2 로그인
- [ ] 일반 로그인 (이메일/비밀번호)
- [ ] Google 로그인
- [ ] 비밀번호 찾기 링크
- [ ] 로그인 성공 시 홈으로 이동
- [ ] 세션 유지

#### 3.3 마이페이지
- [ ] 내 정보 표시
- [ ] 내 게시글 목록
- [ ] 내 댓글 목록
- [ ] 즐겨찾는 정치인 목록
- [ ] 내가 평가한 정치인 목록
- [ ] 프로필 수정 링크

#### 3.4 로그아웃
- [ ] 로그아웃 버튼
- [ ] 세션 종료
- [ ] 홈으로 리다이렉트
