## 📦 기대 결과물

**생성할 파일:**
1. `app/api/cron/recalculate-ranks/route.ts` - Cron API Route
2. `vercel.json` - Cron 스케줄 추가 (기존 파일 업데이트)

**Cron 스케줄:**
- 매일 오전 3시 실행: `0 3 * * *`

**vercel.json 추가:**
```json
{
  "path": "/api/cron/recalculate-ranks",
  "schedule": "0 3 * * *"
}
```

**등급 체계:**
- Level 1 (새싹): 0-99점
- Level 2 (일반): 100-499점
- Level 3 (활동가): 500-1999점
- Level 4 (전문가): 2000-4999점
- Level 5 (명예): 5000점 이상

**포인트 계산 기준:**
- 게시글 작성: +10점
- 댓글 작성: +5점
- 공감 받음: +2점
- 댓글 받음: +3점
- 신고 받음: -50점

**구현해야 할 세부 항목**:

1. Vercel Cron Job 설정 (매일 오전 3시)
2. API Route 생성 (cron 인증 헤더 검증)
3. 사용자 활동 점수 계산 (게시글, 댓글, 공감 합산)
4. 등급 재할당 로직 (포인트에 따른 Level 업데이트)
5. users 테이블 업데이트 (points, level 컬럼)
6. 등급 변경 알림 생성

각 항목을 체계적으로 구현하고 테스트하세요.
