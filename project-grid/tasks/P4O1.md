1. `app/api/cron/update-politicians/route.ts` - Cron API Route
2. `vercel.json` - Vercel Cron 설정 (업데이트)

**Cron 스케줄:**
- 매일 오전 6시 실행: `0 6 * * *`
- 또는 매주 월요일 오전 6시: `0 6 * * 1`

**vercel.json 예시:**
```json
{
  "crons": [{
    "path": "/api/cron/update-politicians",
    "schedule": "0 6 * * *"
  }]
}
```
