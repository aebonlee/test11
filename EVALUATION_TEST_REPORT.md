## 4. V30 평가 기준 적용 검증

### 청렴성(Integrity) 정의
부정 행위, 뇌물, 윤리 위반, 부패 행위 등의 부정적 기록 평가

### 등급별 평가 기준

| 등급 | 판정 | 정의 |
|------|------|------|
| **+4** | 탁월함 | 청렴성이 매우 우수함 - 모범 사례 |
| **+3** | 우수함 | 청렴성이 좋음 - 긍정적 평가 |
| **+2** | 양호함 | 청렴성이 기본 충족 - 적절함 |
| **+1** | 보통 | 청렴성이 평균 수준 - 평범 |
| **-1** | 미흡함 | 청렴성 문제 지적됨 - 개선 필요 |
| **-2** | 부족함 | 청렴성 논란 있음 - 문제 있음 |
| **-3** | 매우 부족 | 청렴성 심각한 문제 - 심각함 |
| **-4** | 극히 부족 | 정치인 부적합 - 심각한 위반 |

### 샘플 평가 사례

#### 긍정 사례 (+3)
```
제목: 취약계층 보호를 위한 건강보험료 징수 개선 법안 발의
근거: 취약계층 보호를 위한 건강보험료 징수 개선 법안 발의는
      공공 복지 증진과 청렴한 의정 활동을 보여줌
```

```
제목: 서초구청장 공약이행 평가 5년 연속 SA(최고등급)
근거: 서초구청장 재직 시 5년 연속 공약이행 최고 등급(SA)을 받은 것은
      뛰어난 청렴성과 투명한 행정 능력을 보여주는 중요한 증거.
      공약 이행의 일관성은 높은 평가를 받을 만함
```

#### 부정 사례 (-2)
```
제목: 정치자금법 위반으로 과태료 부과
근거: 정치자금법 위반으로 과태료 부과는 청렴성에 부정적 영향
```

#### 중립 사례 (+2)
```
제목: 취약계층 복지 개선을 위한 법안 발의
근거: 취약계층 복지 개선을 위한 법안 발의로 사회적 책임감 보여줌
```
