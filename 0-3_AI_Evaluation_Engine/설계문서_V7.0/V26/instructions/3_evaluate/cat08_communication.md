# Category 08: Communication (소통능력) - 평가 지침서 V26.0

---

## 1. 카테고리 정의

### 한글 정의
시민과 효과적으로 의사소통하는 능력

### 영문 정의
The ability to communicate effectively with citizens

### 학술 근거
- **Habermas (1984)**: 의사소통적 합리성 이론
- **Fishkin (2009)**: 숙의 민주주의와 소통
- **Nabatchi (2012)**: 시민 참여와 소통의 관계

---

## 2. 평가 범위

### [O] 소통능력에 해당
- 시민 소통 노력
- SNS 활용 능력
- 언론 대응 능력
- 토론 능력
- 설명/설득 능력
- 경청 자세
- 공감 표현
- 다양한 채널 활용

### [X] 소통능력 아님 (다른 카테고리)
- 정보 공개 → 카테고리 7 (투명성)
- 민원 처리 → 카테고리 9 (대응성)
- 조직 리더십 → 카테고리 2 (리더십)

---

## 3. 등급 체계 (A~H)

| 등급 | 판단 기준 |
|------|-----------|
| **A** | 탁월함 - 소통의 달인, 시민과 공감 |
| **B** | 우수함 - 적극적이고 효과적인 소통 |
| **C** | 양호함 - 소통 노력 인정됨 |
| **D** | 보통 - 기본적인 소통 |
| **E** | 미흡함 - 소통 부족 지적 |
| **F** | 부족함 - 소통 능력 문제 |
| **G** | 매우 부족 - 불통 |
| **H** | 극히 부족 - 완전한 불통 |

---

## 4. 등급 판단 세부 기준

### A등급 (탁월함)
- 소통의 달인으로 인정
- 시민과 적극적 공감
- 다양한 채널 효과적 활용
- 토론/설득 능력 탁월

### B등급 (우수함)
- 적극적이고 효과적인 소통
- SNS 등 채널 활발 운영
- 시민 의견 경청
- 언론 대응 우수

### C등급 (양호함)
- 소통 노력 인정됨
- 기본적인 채널 운영
- 시민 의견 수렴
- 큰 문제 없음

### D등급 (보통)
- 기본적인 소통
- 특별한 노력 없음
- 평균 수준
- 무난한 평가

### E등급 (미흡함)
- 소통 부족 지적
- 채널 운영 소홀
- 시민 의견 경청 부족
- 개선 필요

### F등급 (부족함)
- 소통 능력 문제
- 언론 대응 미흡
- 시민과 거리감
- 불통 비판

### G등급 (매우 부족)
- 불통
- 소통 채널 부재
- 시민 의견 무시
- 심각한 소통 문제

### H등급 (극히 부족)
- 완전한 불통
- 모든 소통 거부
- 시민과 단절
- 정치인 부적합

## 5. 평가 프로세스

### 입력 데이터
```
4개 AI × 50개 = 200개 데이터 (소통능력 카테고리)
```

### 평가 방법
1. 200개 데이터 전체 검토
2. 각 데이터별 등급(A~H) 부여
3. rating 필드에 알파벳 기록

---

## 6. V26.0 기간 제한

### OFFICIAL 데이터: 최근 4년
```
유효 기간: 2022-01-07 ~ 2026-01-07
```

### PUBLIC 데이터: 최근 1년
```
유효 기간: 2025-01-07 ~ 2026-01-07
```

---

## 7. 주의사항 (Pitfalls)

1. **일방 vs 쌍방**: 일방적 홍보 vs 실질적 소통
2. **양 vs 질**: SNS 게시물 수 ≠ 소통 품질
3. **선별 소통**: 특정 집단만 소통하는지 확인
4. **대리 소통**: 본인 소통 vs 보좌관 대리 소통
