| 돈/재산 관련 비위 | 도덕/행위 관련 비위 |
| 뇌물, 횡령, 배임 | 폭언, 막말, 품위손상 |

**기사 게시일 ≠ 사건 발생일**

```

- 2025년 청문회에서 "10년 전 아들 표절" 언급 → 사건은 10년 전 → 수집 금지!
- 2025년 기사: "과거 폭언 논란 재조명" (2015년) → 수집 금지!
- 2024년 보도: "옛날 성희롱 발언" (2018년) → 수집 금지!

✅ 수집 가능 예시:
- 2025년 청문회에서 "무성의한 답변 태도" → 태도가 2025년 → 수집 OK
- 2024년 새로운 폭언 논란 → 사건이 2024년 → 수집 OK
- 청문회 중 "막말" 발생 → 막말이 현재 시점 → 수집 OK

⚠️ 구분 기준:
- "과거 사건 자체"를 보도 → 사건 발생 시점 기준 → 기간 초과면 금지
- "과거 사건에 대한 현재 대응"을 보도 → 대응 시점 기준 → 수집 가능
```



### 🚨🚨 콘텐츠 기반 과거 사건 필터링 (V28.1 필수!) 🚨🚨

**수집 전 내용(content)에서 과거 연도를 반드시 확인!**

```
⛔ 다음 패턴이 내용에 포함되면 절대 수집 금지:

1. 과거 연도 직접 언급 (OFFICIAL: 2022년 미만, PUBLIC: 2024년 미만):
   - "2002년", "2010년", "2015년" 등 기준 초과 연도 → ❌ 금지!
   - "10년 전", "20년 전", "수년 전" → ❌ 금지!
   - "과거", "예전", "당시" + 구체적 사건 → ❌ 금지!

2. 청문회/재조명에서 과거 사건 언급:
   - "청문회에서 ○○년 사건 질의" → ❌ 금지!
   - "과거 ○○ 의혹 재조명" → ❌ 금지!
   - "예전 ○○ 논란 재점화" → ❌ 금지!

3. 과거 판결/확정:
   - "○○년 유죄 확정" → ❌ 금지! (사건 연도 기준)
   - "과거 ○○ 혐의로 처벌" → ❌ 금지!

✅ 수집 가능 - 현재 시점 사건만:
   - "2024년 새로운 의혹 제기" → ✅ OK
   - "2025년 청문회 답변 태도 논란" → ✅ OK
   - "최근 정책 성과 발표" → ✅ OK
```

**핵심: data_content 내용을 읽고, 사건 발생 연도가 기준 외이면 수집하지 마세요!**

### 🚨🚨 과거 사건 언어 패턴 필터링 (V28.2 강화!) 🚨🚨

**연도 없이도 과거 사건임을 알 수 있는 표현 차단!**

```
⛔ 다음 표현이 포함되면 수집 금지 (과거 사건 신호):

1. 진행형/지속 표현 (= 오래된 사건의 증거):
   - "논란이 지속" / "의혹이 지속" / "논란이 계속"
   - "의혹이 계속 제기" / "비판이 이어지고"
   - "여전히 논란" / "아직도 해결되지 않은"
   - "지속적으로 제기되는" / "끊이지 않는 논란"

2. 추가/새로운 표현 (= 기존 사건이 오래됨):
   - "추가 의혹" / "추가 논란" / "추가 정황"
   - "새로운 정황이 드러나" / "새로운 증거"
   - "기존 논란을 넘어서" / "기존에 알려진"
   - "또다시 논란" / "다시 도마 위에"

3. 재조명/환기 표현:
   - "재조명" / "재점화" / "다시 주목"
   - "다시 거론" / "다시 언급" / "재부각"
   - "과거 행적" / "과거 발언" / "과거 이력"

💡 판단 원칙:
"이 표현이 나오면 사건이 처음 발생한 것이 아니다"
→ 처음이 아니면 = 과거 사건 = 기간 검증 대상!
```

### 🚨🚨 중복 수집 강화 규칙 (V28.2 필수!) 🚨🚨

**동일 사건의 다양한 표현을 1건으로 통합!**

```
⛔ 중복 수집 금지 - 핵심 규칙:

1. 동일 주체 + 동일 유형 = 1건만:
   ❌ "A 의혹" + "A 논란" + "A 의혹 지속"
   ✅ 위 3건은 모두 같은 사건 → 1건만 수집!

2. 핵심 키워드 중복 체크:
   - 비위 키워드: 표절, 비리, 횡령, 배임, 뇌물, 정치자금
   - 주체: 본인, 아들, 딸, 배우자, 가족
   - 같은 주체 + 같은 키워드 = 동일 사건!

3. 제목 변형 인식:
   ❌ "A 의혹" = "A 논란" = "A 의혹 제기" = "A 논란 지속"
   ✅ 모두 같은 사건 → 1건만!

4. 후속 보도 = 중복:
   - "○○ 의혹 추가 정황" → 원래 의혹과 동일
   - "○○ 논란 확산" → 원래 논란과 동일
   - "○○ 의혹 재점화" → 원래 의혹과 동일
```

## 8. 주의사항 (Pitfalls)

1. **맥락 고려**: 발언의 전후 맥락 함께 기록
2. **의도 vs 결과**: 의도치 않은 실수 vs 반복적 행위 구분
3. **시기 명시**: 발언/행위 시점 명확히
4. **2차 피해 주의**: 피해자 신원 노출 금지
5. **청렴성과 구분**: 금전적 문제는 청렴성으로

         └→ 검색으로 찾은 실제 데이터만 포함
