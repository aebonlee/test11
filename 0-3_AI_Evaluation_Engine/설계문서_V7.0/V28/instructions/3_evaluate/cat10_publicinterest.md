# Category 10: Public Interest (공익성) - 평가 지침서 V28

---

## 1. 카테고리 정의

### 한글 정의
사익보다 공공의 이익을 우선시하는 자질

### 영문 정의
The quality of prioritizing public interest over private interest

### 학술 근거
- **Bozeman (2007)**: 공익 이론과 측정
- **Frederickson (1997)**: 공공행정의 공익 개념
- **Moore (1995)**: 공공가치 창출 이론

---

## 2. 평가 범위

### [O] 공익성에 해당
- 공공 이익 추구
- 소외계층 배려
- 약자 보호 정책
- 지역 균형 발전
- 환경 보호
- 미래 세대 고려
- 사회적 약자 대변
- 복지 정책

### [X] 공익성 아님 (다른 카테고리)
- 사적 이익 추구 → 카테고리 4 (청렴성)
- 정책 전문성 → 카테고리 1 (전문성)
- 장기 비전 → 카테고리 3 (비전)

## 3. 등급 체계 (+4 ~ -4) - V28

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 공익의 모범, 약자 보호 리더, 소외계층 대변 |
| **+3** | +6 | 우수 - 공익 우선 정치 활동, 복지/환경 정책 관심 |
| **+2** | +4 | 양호 - 공익을 위해 노력, 기본적 약자 배려 |
| **+1** | +2 | 경미한 긍정 - 기본적인 공익 고려, 특별한 노력 없음 |
| **-1** | -2 | 경미한 부정 - 공익 고려 부족, 특정 집단 편향 경향 |
| **-2** | -4 | 일반 부정 - 특정 이익 편향, 지역 이기주의 |
| **-3** | -6 | 심각한 부정 - 사익/특정 이익 우선, 대기업 편향 |
| **-4** | -8 | 최악 - 공익 완전 무시, 노골적 사익 추구 |

---

## 4. 등급 판단 세부 기준

### +4등급 (탁월함)
- 공익의 모범 사례
- 소외계층 적극 대변
- 약자 보호 정책 주도
- 지역/세대 균형 추구

### +3등급 (우수함)
- 공익 우선 정치 활동
- 복지 정책 관심
- 환경 보호 노력
- 사회적 약자 배려

### +2등급 (양호함)
- 공익을 위해 노력
- 기본적인 약자 배려
- 균형 잡힌 정책
- 큰 문제 없음

### +1등급 (경미한 긍정)
- 기본적인 공익 고려
- 특별한 노력 없음
- 평균 수준
- 무난한 평가

### -1등급 (경미한 부정)
- 공익 고려 부족
- 특정 집단 편향 경향
- 약자 배려 미흡
- 개선 필요

### -2등급 (일반 부정)
- 특정 이익 편향
- 지역 이기주의
- 약자 무시 경향
- 공익성 비판

### -3등급 (심각한 부정)
- 사익/특정 이익 우선
- 대기업/특정 집단 편향
- 공익 무시
- 다수의 비판

### -4등급 (최악)
- 공익 완전 무시
- 노골적 사익 추구
- 약자 착취 수준
- 정치인 부적합

## 5. 평가 프로세스

### 입력 데이터
```
50개 (자기가 수집한 데이터) 데이터 (공익성 카테고리)
```

### 평가 방법
1. 50개 데이터 전체 검토
2. 각 데이터별 등급(+4 ~ -4) 부여
3. rating 필드에 숫자 문자열 기록 (예: "+3", "-2")

---

## 6. V28 기간 제한

### OFFICIAL 데이터: 최근 4년
```
유효 기간: 2022-01-07 ~ 2026-01-07
```

### PUBLIC 데이터: 최근 2년
```
유효 기간: 2024-01-11 ~ 2026-01-07
```

## 7. 주의사항 (Pitfalls)

1. **공익 정의 다양성**: 관점에 따라 공익 정의 다름
2. **표면 vs 실질**: 공익적 명분 vs 실제 수혜자
3. **단기 vs 장기**: 단기 인기 정책 vs 장기적 공익
4. **지역 vs 국가**: 지역 이익 vs 국가 전체 이익
