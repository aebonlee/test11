## 3. 수집 규칙

### 카테고리당 50개

```
카테고리 50개 = Phase 1 (10개) + Phase 2 (40개)

Phase 1: 부정 주제 10개 (20% 보장)
├── OFFICIAL 부정 5개 (4년 이내)
└── PUBLIC 부정 5개 (1년 이내)

Phase 2: 자유 평가 40개
├── OFFICIAL 자유 20개 (4년 이내)
└── PUBLIC 자유 20개 (1년 이내)
```

### 기간 제한

| 출처 | 기간 |
|------|------|
| OFFICIAL | 최근 4년 (2022-01-07 ~ 2026-01-07) |
| PUBLIC | 최근 1년 (2025-01-07 ~ 2026-01-07) |

## 6. 점수 계산

### 알파벳 → 숫자 변환

```
A: +8, B: +6, C: +4, D: +2
E: -2, F: -4, G: -6, H: -8
```

### 카테고리 점수 (20~100점)

```python
PRIOR = 6.0
COEFFICIENT = 0.5
category_score = (PRIOR + avg_rating * COEFFICIENT) * 10
```

### 최종 점수 (200~1000점)

```python
total_score = sum(10개 카테고리 점수)
# 범위: 200 ~ 1000점
```
