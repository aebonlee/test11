# Category 05: Ethics (윤리성) - 평가 지침서 V27.0

---

## 1. 카테고리 정의

### 한글 정의
사회적 규범과 도덕적 품위를 유지하는 자질

### 영문 정의
The quality of maintaining social norms and moral dignity

### 학술 근거
- **Aristotle**: 덕 윤리학 (Virtue Ethics)
- **Kohlberg (1981)**: 도덕 발달 이론
- **Rest (1986)**: 도덕적 의사결정 모델

---

## 2. 평가 범위

### [O] 윤리성에 해당
- 도덕성
- 가치관
- 인격
- 사회적 책임
- 공정성
- 차별 없는 태도
- 성평등
- 인권 존중
- 발언의 적절성
- 품위 유지
- 성범죄 (비금전적)
- 폭언/막말
- 혐오 발언

### [X] 윤리성 아님 (다른 카테고리)
- 금품 수수/뇌물 → 카테고리 4 (청렴성)
- 공약 불이행 → 카테고리 6 (책임성)
- 정보 은폐 → 카테고리 7 (투명성)

## 3. 등급 체계 (A~H)

| 등급 | 판단 기준 |
|------|-----------|
| **A** | 탁월함 - 도덕적 모범, 인권 존중 리더 |
| **B** | 우수함 - 품위 있는 정치 활동 |
| **C** | 양호함 - 윤리적으로 문제 없음 |
| **D** | 보통 - 특별한 문제 없음 |
| **E** | 미흡함 - 경미한 윤리적 문제 |
| **F** | 부족함 - 윤리성 논란 존재 |
| **G** | 매우 부족 - 심각한 윤리적 문제 |
| **H** | 극히 부족 - 중대한 비윤리적 행위 |

---

## 4. 등급 판단 세부 기준

### A등급 (탁월함)
- 도덕적 모범 사례
- 인권 존중 활동 주도
- 사회적 약자 옹호
- 품위 있는 정치 문화 선도

### B등급 (우수함)
- 품위 있는 정치 활동
- 혐오/차별 발언 없음
- 존중의 자세 유지
- 윤리적 리더십

### C등급 (양호함)
- 전반적으로 윤리적
- 경미한 실수 정도
- 품위 유지
- 윤리적 비판 적음

### D등급 (보통)
- 특별한 윤리적 문제 없음
- 평범한 수준
- 주목할 이슈 없음
- 무난한 평가

### E등급 (미흡함)
- 경미한 윤리적 문제
- 부적절한 발언 경험
- 사과 후 개선
- 개선 필요

### F등급 (부족함)
- 윤리성 논란 존재
- 막말/폭언 사례
- 차별적 발언 지적
- 품위 손상

### G등급 (매우 부족)
- 심각한 윤리적 문제
- 반복적 막말/폭언
- 혐오 발언
- 성희롱 의혹

### H등급 (극히 부족)
- 중대한 비윤리적 행위
- 성범죄 확정
- 심각한 인권 침해
- 정치인 부적합

## 5. 평가 프로세스

### 입력 데이터
```
50개 (자기가 수집한 데이터) 데이터 (윤리성 카테고리)
```

### 평가 방법
1. 50개 데이터 전체 검토
2. 각 데이터별 등급(A~H) 부여
3. rating 필드에 알파벳 기록

유효 기간: 2022-01-07 ~ 2026-01-07

유효 기간: 2025-01-07 ~ 2026-01-07

## 7. 주의사항 (Pitfalls)

1. **맥락 고려**: 발언의 전후 맥락 함께 고려
2. **의도 vs 결과**: 의도치 않은 실수 vs 반복적 행위
3. **시기 명시**: 발언/행위 시점 확인
4. **2차 피해 주의**: 피해자 신원 노출 주의
5. **청렴성과 구분**: 금전적 문제는 청렴성으로
