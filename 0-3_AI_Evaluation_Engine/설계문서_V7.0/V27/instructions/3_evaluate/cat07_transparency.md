# Category 07: Transparency (투명성) - 평가 지침서 V27.0

---

## 1. 카테고리 정의

### 한글 정의
정보를 숨김없이 공개하고 공유하는 자질

### 영문 정의
The quality of openly disclosing and sharing information without concealment

### 학술 근거
- **Florini (2007)**: 투명성의 민주적 가치
- **Hood (2006)**: 정부 투명성 측정 지표
- **Grimmelikhuijsen (2012)**: 투명성과 신뢰의 관계

---

## 2. 평가 범위

### [O] 투명성에 해당
- 정보 공개 정도
- 정책 결정 과정 공개
- 재정 집행 투명성
- 의사결정 근거 설명
- 회의록/기록 공개
- 언론 인터뷰 응대
- 자료 제출 성실성

### [X] 투명성 아님 (다른 카테고리)
- 금전 비리 → 카테고리 4 (청렴성)
- 시민 소통 → 카테고리 8 (소통능력)
- 민원 대응 → 카테고리 9 (대응성)

---

## 3. 등급 체계 (A~H)

| 등급 | 판단 기준 |
|------|-----------|
| **A** | 탁월함 - 투명성의 모범, 적극적 정보 공개 |
| **B** | 우수함 - 정보 공개에 적극적 |
| **C** | 양호함 - 투명하게 활동 |
| **D** | 보통 - 기본적인 투명성 유지 |
| **E** | 미흡함 - 일부 정보 공개 소극적 |
| **F** | 부족함 - 투명성 문제 제기됨 |
| **G** | 매우 부족 - 정보 은폐 경향 |
| **H** | 극히 부족 - 심각한 불투명 |

---

## 4. 등급 판단 세부 기준

### A등급 (탁월함)
- 투명성의 모범 사례
- 적극적 정보 공개 주도
- 재정 완전 투명 공개
- 시민단체 투명성 평가 최우수

### B등급 (우수함)
- 정보 공개에 적극적
- 정책 결정 과정 설명
- 재정 투명하게 운영
- 언론 질의에 성실 대응

### C등급 (양호함)
- 전반적으로 투명하게 활동
- 정보 공개 요청에 응답
- 기본적인 투명성 유지
- 큰 문제 없음

### D등급 (보통)
- 기본적인 투명성
- 특별한 문제 없음
- 평균 수준
- 무난한 평가

### E등급 (미흡함)
- 일부 정보 공개 소극적
- 비공개 결정 사례 있음
- 투명성 개선 필요
- 약간의 비판

### F등급 (부족함)
- 투명성 문제 제기됨
- 정보 공개 거부 사례
- 시민단체 비판
- 불투명 경향

### G등급 (매우 부족)
- 정보 은폐 경향
- 다수의 비공개 결정
- 자료 제출 거부
- 심각한 불투명

### H등급 (극히 부족)
- 심각한 불투명
- 조직적 정보 은폐
- 기록 삭제/조작
- 정치인 부적합

---

## 5. 평가 프로세스

### 입력 데이터
```
50개 (자기가 수집한 데이터) 데이터 (투명성 카테고리)
```

### 평가 방법
1. 50개 데이터 전체 검토
2. 각 데이터별 등급(A~H) 부여
3. rating 필드에 알파벳 기록

---

## 6. V27.0 기간 제한

### OFFICIAL 데이터: 최근 4년
```
유효 기간: 2022-01-07 ~ 2026-01-07
```

### PUBLIC 데이터: 최근 1년
```
유효 기간: 2025-01-07 ~ 2026-01-07
```

---

## 7. 주의사항 (Pitfalls)

1. **법적 비공개**: 법적으로 비공개 가능한 정보 구분
2. **공개 범위**: 부분 공개 vs 전체 공개
3. **적시성**: 정보 공개 시점 (요청 전/후)
4. **형식적 공개**: 형식만 갖춘 공개 vs 실질적 공개
