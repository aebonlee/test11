## ✅ 검증 항목 (5가지)

### 검증 1: 수집-평가 10개 항목 완전 일치

**검증 내용**:
각 카테고리의 수집 지침서와 평가 지침서에 있는 "구체적 10개 항목"이 **100% 동일**한지 확인

**예시 - 전문성 카테고리**:

수집 지침서 (`2_collect/cat01_expertise.md`):
```markdown
### ✅ [O] 전문성 수집 기준 10개 항목

| # | 항목명 | 설명 |
|---|--------|------|
| 1-1 | **최종 학력 수준** | 박사/석사/학사/전문대 학위 |
| 1-2 | **직무 관련 자격증** | 변호사, CPA, 교원자격증 등 |
...
| 1-10 | **전문성 관련 언론 평가** | 전문성 키워드 보도 |
```

평가 지침서 (`3_evaluate/cat01_expertise.md`):
```markdown
### ✅ [O] 전문성 평가 기준 10개 항목

| # | 항목명 | 설명 |
|---|--------|------|
| 1-1 | **최종 학력 수준** | 박사/석사/학사/전문대 학위 |
| 1-2 | **직무 관련 자격증** | 변호사, CPA, 교원자격증 등 |
...
| 1-10 | **전문성 관련 언론 평가** | 전문성 키워드 보도 |
```

**✅ 통과 기준**: 10개 항목의 번호, 항목명, 설명이 **완전히 동일**

**검증 방법**:
1. 수집 지침서에서 테이블 추출
2. 평가 지침서에서 테이블 추출
3. 항목별 문자열 비교 (대소문자, 공백 포함)

## 🔧 자동 검증 스크립트

다음 Python 스크립트를 실행하여 자동 검증할 수 있습니다:

```bash
python3 << 'EOF'
import re
import os

categories = [
    ("cat01_expertise.md", 1, "전문성"),
    ("cat02_leadership.md", 2, "리더십"),
    ("cat03_vision.md", 3, "비전"),
    ("cat04_integrity.md", 4, "청렴성"),
    ("cat05_ethics.md", 5, "윤리성"),
    ("cat06_accountability.md", 6, "책임감"),
    ("cat07_transparency.md", 7, "투명성"),
    ("cat08_communication.md", 8, "소통능력"),
    ("cat09_responsiveness.md", 9, "대응성"),
    ("cat10_publicinterest.md", 10, "공익성")
]

collect_base = "설계문서_V7.0/instructions_v30/2_collect/"
eval_base = "설계문서_V7.0/instructions_v30/3_evaluate/"

print("=" * 80)
print("V30 문서 일관성 자동 검증")
print("=" * 80)

# 검증 1: 10개 항목 일치
print("\n[검증 1] 수집-평가 10개 항목 일치")
success_count = 0
for filename, cat_num, kor_name in categories:
    collect_path = os.path.join(collect_base, filename)
    with open(collect_path, 'r', encoding='utf-8') as f:
        collect_items = re.findall(r'\| (\d+-\d+) \| \*\*(.*?)\*\* \|', f.read())

    eval_path = os.path.join(eval_base, filename)
    with open(eval_path, 'r', encoding='utf-8') as f:
        eval_items = re.findall(r'\| (\d+-\d+) \| \*\*(.*?)\*\* \|', f.read())

    if collect_items == eval_items and len(collect_items) == 10:
        print(f"  OK {kor_name}")
        success_count += 1
    else:
        print(f"  FAIL {kor_name}: 불일치!")

print(f"결과: {success_count}/10 통과")

# 검증 2: V30 표기
print("\n[검증 2] V30 버전 표기")
v30_count = 0
for filename, cat_num, kor_name in categories:
    for base in [collect_base, eval_base]:
        path = os.path.join(base, filename)
        with open(path, 'r', encoding='utf-8') as f:
            if "V30" in f.read():
                v30_count += 1

print(f"결과: {v30_count}/20 파일이 V30 사용")

# 검증 3: 기간 제한
print("\n[검증 3] 기간 제한 일관성")
date_count = 0
for filename, cat_num, kor_name in categories:
    for base in [collect_base, eval_base]:
        path = os.path.join(base, filename)
        with open(path, 'r', encoding='utf-8') as f:
            content = f.read()
            if "최근 4년" in content and "최근 2년" in content:
                date_count += 1

print(f"결과: {date_count}/20 파일이 상대적 기간 표현 사용")

# 최종 결과
print("\n" + "=" * 80)
if success_count == 10 and v30_count == 20 and date_count == 20:
    print("✅ 모든 검증 통과!")
else:
    print("⚠️ 일부 검증 실패")
    print(f"  - 항목 일치: {success_count}/10")
    print(f"  - V30 표기: {v30_count}/20")
    print(f"  - 기간 표현: {date_count}/20")
print("=" * 80)
EOF
```
