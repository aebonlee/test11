## 1. 카테고리 정의

### 한글 정의
사회적 규범과 도덕적 품위를 유지하는 자질

### 영문 정의
The quality of maintaining social norms and moral dignity

### 학술 근거
- **Aristotle**: 덕 윤리학 (Virtue Ethics)
- **Kohlberg (1981)**: 도덕 발달 이론
- **Rest (1986)**: 도덕적 의사결정 모델

## 4. 등급 체계 (+4 ~ -4) - V30

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 도덕적 모범, 인권 존중 리더, 사회적 약자 옹호 |
| **+3** | +6 | 우수 - 품위 있는 정치 활동, 혐오/차별 발언 없음 |
| **+2** | +4 | 양호 - 윤리적으로 문제 없음, 품위 유지 |
| **+1** | +2 | 경미한 긍정 - 특별한 문제 없음, 무난함 |
| **0** | 0 | 중립 - 긍정/부정 판단 불가, 정보 부족 또는 중립적 |
| **-1** | -2 | 경미한 부정 - 경미한 윤리적 문제, 사과 후 개선 |
| **-2** | -4 | 일반 부정 - 윤리성 논란, 막말/폭언 사례 |
| **-3** | -6 | 심각한 부정 - 심각한 윤리적 문제, 성희롱 의혹 |
| **-4** | -8 | 최악 - 중대한 비윤리적 행위, 성범죄 확정 |

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 도덕적 모범 사례
- 인권 존중 활동 주도
- 사회적 약자 옹호
- 품위 있는 정치 문화 선도

### +3등급 (우수함)
- 품위 있는 정치 활동
- 혐오/차별 발언 없음
- 존중의 자세 유지
- 윤리적 리더십

### +2등급 (양호함)
- 전반적으로 윤리적
- 경미한 실수 정도
- 품위 유지
- 윤리적 비판 적음

### +1등급 (경미한 긍정)
- 특별한 윤리적 문제 없음
- 평범한 수준
- 주목할 이슈 없음
- 무난한 평가

### -1등급 (경미한 부정)
- 경미한 윤리적 문제
- 부적절한 발언 경험
- 사과 후 개선
- 개선 필요

### -2등급 (일반 부정)
- 윤리성 논란 존재
- 막말/폭언 사례
- 차별적 발언 지적
- 품위 손상

### -3등급 (심각한 부정)
- 심각한 윤리적 문제
- 반복적 막말/폭언
- 혐오 발언
- 성희롱 의혹

### -4등급 (최악)
- 중대한 비윤리적 행위
- 성범죄 확정
- 심각한 인권 침해
- 정치인 부적합

## 6. 평가 프로세스

### 입력 데이터
```
100개 풀링 데이터 (윤리성 카테고리)
- Gemini 수집: 60개
- Perplexity 수집: 30개
- Grok 수집: 10개
```

### 평가 방법
1. **100개 풀링 데이터 전체 검토**
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 AI와 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **맥락 고려**: 발언의 전후 맥락 함께 고려
2. **의도 vs 결과**: 의도치 않은 실수 vs 반복적 행위
3. **시기 명시**: 발언/행위 시점 확인
4. **2차 피해 주의**: 피해자 신원 노출 주의
5. **청렴성과 구분**: 금전적 문제는 청렴성으로
