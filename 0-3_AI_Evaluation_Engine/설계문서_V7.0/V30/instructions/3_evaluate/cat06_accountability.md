# Category 06: Accountability (책임성) - 평가 지침서 V30

---

## 1. 카테고리 정의

### 한글 정의
맡은 직무와 약속에 대해 책임지는 자질

### 영문 정의
The quality of taking responsibility for assigned duties and promises

### 학술 근거
- **Bovens (2010)**: 공적 책임성의 3요소
- **Mulgan (2000)**: 민주적 책임성 이론
- **Schedler (1999)**: 책임성의 두 축


## 2. V30 평가 방식 - 풀링 평가

### 평가 대상 데이터

```

```

### 평가 독립성

```
타이밍 분리 = 객관성 보장

[수집 단계]
→ Gemini, Perplexity, Grok이 데이터 수집
→ collected_data_v30 테이블에 저장

[평가 단계] - 다른 세션, 다른 시점
→ Claude, ChatGPT, Gemini, Grok이 평가
→ 수집 시점 ≠ 평가 시점
→ 독립적 판단 = 더 객관적
```

⚠️ **아래 10개 항목을 기준으로 평가하세요.**

## 4. 등급 체계 (+4 ~ -4) - V30

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 공약 이행률 90% 이상, 책임감 모범 |
| **+3** | +6 | 우수 - 공약 대부분 이행 (70-89%), 출석률 우수 |
| **+2** | +4 | 양호 - 공약 이행 노력 (50-69%), 직무 충실 |
| **+1** | +2 | 경미한 긍정 - 기본적인 책임 이행 |
| **0** | 0 | 중립 - 긍정/부정 판단 불가, 정보 부족 또는 중립적 |
| **-1** | -2 | 경미한 부정 - 일부 공약 미이행, 책임 회피 경향 |
| **-2** | -4 | 일반 부정 - 공약 이행률 저조 (30% 미만) |
| **-3** | -6 | 심각한 부정 - 심각한 무책임, 공약 대부분 파기 |
| **-4** | -8 | 최악 - 직무 유기 수준, 공약 완전 파기 |

---

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 공약 이행률 90% 이상
- 출석률 95% 이상
- 책임감 있는 리더로 인정
- 실패 시에도 책임지는 자세

### +3등급 (우수함)
- 공약 대부분 이행 (70-89%)
- 출석률 우수 (90% 이상)
- 직무에 충실
- 후속 조치 이행

### +2등급 (양호함)
- 공약 이행 노력 확인 (50-69%)
- 출석률 양호 (80% 이상)
- 기본적인 책임감
- 큰 문제 없음

### +1등급 (경미한 긍정)
- 기본적인 책임 이행
- 평균 수준의 공약 이행
- 특별한 문제 없음
- 무난한 평가

### -1등급 (경미한 부정)
- 일부 공약 미이행
- 출석률 저조한 경우 있음
- 책임 회피 경향
- 개선 필요

### -2등급 (일반 부정)
- 공약 이행률 저조 (30% 미만)
- 출석률 문제
- 무책임 비판
- 직무 태만 지적

### -3등급 (심각한 부정)
- 심각한 무책임
- 공약 대부분 파기
- 직무 유기 수준
- 다수의 비판

### -4등급 (최악)
- 직무 유기 확정
- 공약 완전 파기
- 책임 완전 회피
- 정치인 부적합

## 6. 평가 프로세스

### 입력 데이터
```
100개 풀링 데이터 (책임감 카테고리)
- Gemini 수집: 60개
- Perplexity 수집: 30개
- Grok 수집: 10개
```

### 평가 방법
1. **100개 풀링 데이터 전체 검토**
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 AI와 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **환경 변화 고려**: 외부 요인으로 공약 수정 불가피한 경우
2. **정량/정성 구분**: 이행률 수치와 질적 성과 함께
3. **대안 제시**: 불이행 시 대안 제시 여부
4. **출석률 맥락**: 해외 출장 등 정당한 사유
