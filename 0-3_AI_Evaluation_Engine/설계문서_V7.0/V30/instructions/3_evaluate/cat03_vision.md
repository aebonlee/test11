# Category 03: Vision (비전) - 평가 지침서 V30

## 1. 카테고리 정의

### 한글 정의
미래를 예측하고 장기적 목표를 제시하는 능력

### 영문 정의
The ability to predict the future and present long-term goals

### 학술 근거
- **Kouzes & Posner (2017)**: 비전 제시는 리더의 핵심 역량
- **Senge (1990)**: 공유 비전의 중요성
- **Collins & Porras (1996)**: 비전 기업의 성공 요인


## 2. V30 평가 방식 - 풀링 평가

### 평가 대상 데이터

```
100개 풀링 데이터 (3개 AI가 수집한 전체 데이터)

┌──────────────────────────────────────┐
│ Gemini    수집: 60개                 │
│ Perplexity 수집: 30개                │
│ Grok       수집: 10개                │
├──────────────────────────────────────┤
│ 풀링 합계: 100개                     │
└──────────────────────────────────────┘

4개 평가 AI (Claude, ChatGPT, Gemini, Grok)가
각각 100개 전체 데이터를 독립적으로 평가

⚠️ Perplexity = 평가 제외 (수집만 담당)
```

### 평가 독립성

```
타이밍 분리 = 객관성 보장

[수집 단계]
→ Gemini, Perplexity, Grok이 데이터 수집
→ collected_data_v30 테이블에 저장

[평가 단계] - 다른 세션, 다른 시점
→ Claude, ChatGPT, Gemini, Grok이 평가
→ 수집 시점 ≠ 평가 시점
→ 독립적 판단 = 더 객관적
```

## 4. 등급 체계 (+4 ~ -4) - V30

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 시대를 앞서가는 비전 제시, 검증된 예측 |
| **+3** | +6 | 우수 - 구체적이고 실현 가능한 미래 계획 |
| **+2** | +4 | 양호 - 평균 이상의 미래 지향성 |
| **+1** | +2 | 경미한 긍정 - 기본적인 장기 계획 제시 |
| **0** | 0 | 중립 - 긍정/부정 판단 불가, 정보 부족 또는 중립적 |
| **-1** | -2 | 경미한 부정 - 비전 제시 능력 부족 |
| **-2** | -4 | 일반 부정 - 미래 전략 결여 |
| **-3** | -6 | 심각한 부정 - 구태의연한 접근, 혁신 실패 |
| **-4** | -8 | 최악 - 비전 완전 부재, 퇴행적 정책 |

---

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 시대를 앞서가는 혁신적 비전 제시
- 제시한 비전이 실현되어 검증된 경우
- 국가/지역 발전 방향에 영향력
- 다른 정치인들이 따르는 비전 리더

### +3등급 (우수함)
- 구체적이고 실현 가능한 장기 계획
- 혁신 정책 추진 성과
- 시대 변화에 선제적 대응
- 명확한 청사진 제시

### +2등급 (양호함)
- 평균 이상의 미래 지향적 사고
- 장기 계획 수립 능력
- 변화에 대한 적응력
- 일정 수준의 혁신성

### +1등급 (경미한 긍정)
- 기본적인 장기 계획 제시
- 특별히 혁신적이지 않음
- 무난한 수준의 미래 대응
- 현상 유지 수준

### -1등급 (경미한 부정)
- 비전 제시 능력 부족
- 단기적 사고 경향
- 변화 대응 미흡
- 장기 계획 불명확

### -2등급 (일반 부정)
- 미래 전략 결여
- 과거 지향적 접근
- 변화 거부 성향
- 비전 부재 비판

### -3등급 (심각한 부정)
- 구태의연한 접근
- 시대 흐름 역행
- 혁신 실패 사례 다수
- 미래 대비 전무

### -4등급 (최악)
- 비전 완전 부재
- 퇴행적 정책 추구
- 모든 평가에서 최하위
- 미래 준비 불가

## 6. 평가 프로세스

### 입력 데이터
```
100개 풀링 데이터 (비전 카테고리)
- Gemini 수집: 60개
- Perplexity 수집: 30개
- Grok 수집: 10개
```

### 평가 방법
1. **100개 풀링 데이터 전체 검토**
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 AI와 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **공약 ≠ 비전**: 단순 공약과 장기 비전 구분
2. **실현 가능성**: 허황된 약속 vs 실현 가능한 비전
3. **시대 맥락**: 당시 상황에서의 혁신성 평가
4. **구체성**: 추상적 구호 vs 구체적 계획
