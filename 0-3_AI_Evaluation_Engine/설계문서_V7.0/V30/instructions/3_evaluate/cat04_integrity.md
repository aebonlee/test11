## 1. 카테고리 정의

### 한글 정의
금전적/물질적 부패를 저지르지 않는 자질

### 영문 정의
The quality of not engaging in financial or material corruption

### 학술 근거
- **Stoker (2024)**: Competence-Integrity-Authenticity Framework
- **Warren (2004)**: 민주주의의 핵심 - 부패 방지
- **Transparency International**: 청렴성 측정 지표

## 4. 등급 체계 (+4 ~ -4) - V30

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 청렴의 모범, 부패 척결 주도, 수상 |
| **+3** | +6 | 우수 - 깨끗한 정치 활동, 시민단체 청렴 평가 우수 |
| **+2** | +4 | 양호 - 청렴하게 활동, 금전적 비리 없음 |
| **+1** | +2 | 경미한 긍정 - 특별한 문제 없음, 무난함 |
| **0** | 0 | 중립 - 긍정/부정 판단 불가, 정보 부족 또는 중립적 |
| **-1** | -2 | 경미한 부정 - 경미한 의혹, 해명 후 무혐의 |
| **-2** | -4 | 일반 부정 - 청렴성 문제 제기, 수사/조사 대상 |
| **-3** | -6 | 심각한 부정 - 심각한 비리 의혹, 기소 |
| **-4** | -8 | 최악 - 유죄 확정, 중대한 금전적 비리 |

---

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 청렴의 모범 사례로 인정
- 부패 척결 활동 주도
- 재산 공개 모범적
- 시민단체 청렴 평가 최우수

### +3등급 (우수함)
- 깨끗한 정치 활동 유지
- 비리 의혹 전혀 없음
- 정치자금 투명하게 운영
- 공직자 윤리 준수

### +2등급 (양호함)
- 전반적으로 청렴하게 활동
- 경미한 절차적 이슈만 존재
- 금전적 비리 없음
- 청렴 평가 양호

### +1등급 (경미한 긍정)
- 특별한 청렴성 문제 없음
- 평범한 수준의 청렴도
- 주목할 만한 이슈 없음
- 무난한 평가

### -1등급 (경미한 부정)
- 경미한 의혹 제기된 적 있음
- 해명 후 무혐의 처리
- 절차적 위반 지적
- 개선 필요 사항 존재

### -2등급 (일반 부정)
- 청렴성 문제 공식 제기됨
- 수사/조사 대상이 된 적 있음
- 시민단체 비판
- 정치자금 관련 의혹

### -3등급 (심각한 부정)
- 심각한 비리 의혹
- 기소된 적 있음
- 다수의 비리 의혹
- 재산 은닉 의혹

### -4등급 (최악)
- 유죄 확정
- 중대한 금전적 비리
- 횡령/배임 확정
- 정치인으로서 부적합

## 6. 평가 프로세스

### 입력 데이터
```
100개 풀링 데이터 (청렴성 카테고리)
- Gemini 수집: 60개
- Perplexity 수집: 30개
- Grok 수집: 10개
```

### 평가 방법
1. **100개 풀링 데이터 전체 검토**
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 AI와 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **의혹 vs 확정 구분**: 수사 중 vs 유죄 확정 명확히 구분
2. **정치적 공격 구분**: 정쟁성 공격 vs 실제 비리 구별
3. **금액 규모 고려**: 비리 금액 규모에 따른 차등
4. **시기 명시**: 사건 발생 시기 확인
5. **결과 추적**: 수사/재판 결과 확인
