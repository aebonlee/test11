## 1. 카테고리 정의

### 한글 정의
조직과 사람을 효과적으로 이끌어 목표를 달성하는 능력

### 영문 정의
The ability to effectively lead organizations and people to achieve goals

### 학술 근거
- **Stoker (2024)**: 리더십은 조직 성과의 핵심 동인
- **Northouse (2021)**: 영향력을 통한 목표 달성 과정
- **Bass & Riggio (2006)**: 변혁적 리더십 이론

## 2. V30 평가 방식 - 풀링 평가

### 평가 대상 데이터

```
100개 풀링 데이터 (3개 AI가 수집한 전체 데이터)

┌──────────────────────────────────────┐
│ Gemini    수집: 60개                 │
│ Perplexity 수집: 30개                │
│ Grok       수집: 10개                │
├──────────────────────────────────────┤
│ 풀링 합계: 100개                     │
└──────────────────────────────────────┘

4개 평가 AI (Claude, ChatGPT, Gemini, Grok)가
각각 100개 전체 데이터를 독립적으로 평가

⚠️ Perplexity = 평가 제외 (수집만 담당)
```

### 평가 독립성

```
타이밍 분리 = 객관성 보장

[수집 단계]
→ Gemini, Perplexity, Grok이 데이터 수집
→ collected_data_v30 테이블에 저장

[평가 단계] - 다른 세션, 다른 시점
→ Claude, ChatGPT, Gemini, Grok이 평가
→ 수집 시점 ≠ 평가 시점
→ 독립적 판단 = 더 객관적
```

## 4. 등급 체계 (+4 ~ -4) - V30

**숫자 기반 명확한 기준**

| 등급 | 점수(×2) | 객관적 기준 |
|------|----------|-------------|
| **+4** | +8 | 탁월 - 조직을 성공으로 이끈 검증된 리더, 위기 극복 |
| **+3** | +6 | 우수 - 위기 상황에서도 안정적 리더십, 성과 입증 |
| **+2** | +4 | 양호 - 조직 관리 능력 평균 이상, 무난한 운영 |
| **+1** | +2 | 경미한 긍정 - 기본적인 리더십 역량, 노력 확인 |
| **0** | 0 | 중립 - 긍정/부정 판단 불가, 정보 부족 또는 중립적 |
| **-1** | -2 | 경미한 부정 - 리더십 발휘에 어려움, 비판 일부 |
| **-2** | -4 | 일반 부정 - 조직 관리 능력 부족, 갈등 유발 |
| **-3** | -6 | 심각한 부정 - 심각한 리더십 문제, 조직 혼란 |
| **-4** | -8 | 최악 - 리더 부적합, 조직 와해 책임 |

## 5. 등급 판단 세부 기준

### +4등급 (탁월함)
- 조직을 성공적으로 이끈 명확한 실적
- 위기 상황에서 탁월한 대응 사례
- **국무총리/부총리/장관급** 직책에서의 국가 위기 대응 성과
- 당 대표/원내대표로서 당 운영 성공 사례
- 동료/부하 직원의 높은 신뢰

### +3등급 (우수함)
- 조직 관리에서 우수한 성과
- 위기 대응 능력 검증
- 정책 추진력 우수
- 갈등 조정 능력 인정

### +2등급 (양호함)
- 평균 이상의 조직 관리 능력
- 팀워크 형성 능력 양호
- 의사결정에 큰 문제 없음
- 리더십 관련 비판 적음

### +1등급 (경미한 긍정)
- 기본적인 리더십 역량
- 특별한 성과나 실패 없음
- 무난한 조직 운영
- 평범한 수준의 영향력

### -1등급 (경미한 부정)
- 리더십 발휘 어려움 사례 있음
- 조직 내 갈등 관리 미흡
- 의사결정 지연/우유부단
- 리더십 비판 일부 존재

### -2등급 (일반 부정)
- 조직 관리 실패 사례
- 위기 대응 미흡
- 내부 갈등 유발
- 리더십 부재 비판 다수

### -3등급 (심각한 부정)
- 심각한 리더십 실패
- 조직 와해 책임
- 다수의 리더십 비판
- 의사결정 능력 결여

### -4등급 (최악)
- 리더로서 완전히 부적합
- 조직 파괴적 행동
- 법적/도덕적 문제로 리더십 상실
- 모든 평가에서 최하위

## 6. 평가 프로세스

### 입력 데이터
```
100개 풀링 데이터 (리더십 카테고리)
- Gemini 수집: 60개
- Perplexity 수집: 30개
- Grok 수집: 10개
```

### 평가 방법
1. **100개 풀링 데이터 전체 검토**
2. **각 데이터별 등급(+4 ~ -4) 부여**
3. **rating 필드에 숫자 문자열 기록 (예: "+3", "-2")**

### 평가 독립성
- 수집 AI와 평가 AI는 시점이 분리됨
- 각 평가 AI는 독립적으로 100개 전체 평가
- 자신이 수집한 데이터도 객관적으로 재평가

## 8. 주의사항 (Pitfalls)

1. **직책과 리더십의 관계**: 직책만으로 판단하지 말되, **고위 직책에서의 실질적 성과는 높이 평가**
   - 국무총리, 부총리, 장관급 직책에서의 위기 대응 → +4 가능
   - 당 대표/원내대표로서 당 운영 성과 → +4 가능
   - 단순 직책 보유만으로는 +4 불가, 실질적 성과 필요
2. **정쟁성 비판 구분**: 정치적 공격 vs 실제 리더십 문제
3. **단기 vs 장기**: 일시적 위기 vs 지속적 리더십 문제
4. **맥락 고려**: 당시 상황과 제약 조건 함께 고려
