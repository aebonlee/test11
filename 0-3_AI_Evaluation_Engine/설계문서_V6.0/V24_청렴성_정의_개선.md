# V24.0 청렴성(Integrity) 정의 개선

**작성일**: 2025-11-24
**버전**: V24.0
**작업 내용**: 청렴성 카테고리 인스트럭션 개선

---

## 📋 문제 인식

### 데이터 품질 검토 결과
- **한동훈**: 청렴성 정의 일치율 78.0% (39/50개)
- **이준석**: 청렴성 정의 일치율 54.0% (27/50개)

### 주요 문제점
1. **정의 범위 불명확**: "부패 방지, 윤리 준수, 이해충돌 회피"만으로는 경계가 모호
2. **다른 카테고리와 혼동**:
   - 정치적 중립성 → 청렴성이 아닌 책임성(6)
   - 당내 갈등/선거 → 청렴성이 아닌 리더십(2)
   - 정책 전문성 → 청렴성이 아닌 전문성(1)
   - 당적 변경 → 청렴성이 아닌 책임성(6)

3. **Claude API의 광범위한 해석**:
   - 영어 "Integrity"는 한국어 "청렴성"보다 의미가 넓음
   - 도덕적 완전성, 일관성, 신뢰성까지 포함하는 경향

## ✅ 개선 내용

### 기존 정의 (V24.0 이전)
```
청렴성: 부패 방지, 윤리 준수, 이해충돌 회피
```

### 개선된 정의 (V24.0)
```
청렴성: 부패 방지, 윤리 준수, 이해충돌 회피

✅ 청렴성에 해당하는 항목:
- 금품/뇌물 수수, 횡령, 배임, 비리
- 공직자 윤리 위반, 행동강령 위반
- 이해충돌, 겸직 문제, 사적 이익 추구
- 불법 정치자금, 선거법 위반
- 직권 남용, 부적절한 청탁

❌ 청렴성이 아닌 항목 (다른 카테고리):
- 정치적 중립성 → 책임성(6)
- 정치적 편향성 → 윤리성(5)
- 당내 갈등/당대표 선거 → 리더십(2)
- 정책 전문성 부족 → 전문성(1)
- 당적 변경 → 책임성(6) 또는 윤리성(5)
- 법적 책임 회피 → 책임성(6)
- 정보 공개 미흡 → 투명성(7)
```
