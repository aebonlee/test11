# V24.0 등급 체계 개선안

**작성일**: 2025-11-20
**목적**: V23.0의 혼란스러운 등급 체계 개선

---

## 🎯 문제점 (V23.0)

### 현재 등급 체계
```
A, B, C, D, -D, -C, -B, -A
```

**문제점**:
1. **헷갈림**: -A가 A보다 나쁜지, A가 -A보다 나쁜지 직관적이지 않음
2. **역순 혼란**: -A는 가장 나쁜 등급인데, A는 가장 좋은 등급
3. **마이너스 기호**: 시각적으로 혼란스러움

---

## ✅ 개선안 (V24.0)

### 새로운 등급 체계
```
A B C D E F G H (8등급)
```

**장점**:
1. **직관적**: A → H로 갈수록 나빠짐 (학교 성적과 동일)
2. **명확한 순서**: 알파벳 순서대로
3. **헷갈림 없음**: 마이너스 기호 제거
