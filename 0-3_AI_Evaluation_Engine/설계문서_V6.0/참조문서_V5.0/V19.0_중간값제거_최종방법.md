## 📚 학술적 근거

### 1. Forced-Choice Scale

**정의:**
```
"A rating system that does not allow for Undecided,
Neutral, Doesn't know, or No opinion response"
```

**효과:**
- 명확한 의견 강제
- Central Tendency Bias 완화
- 변별력 증가

**출처:** 복수 학술 연구, Survey Methodology

---

### 2. Central Tendency Bias 완화

**문제:**
```
"When a Likert scale includes a 'neutral' or 'safe' option,
a central tendency bias phenomenon occurs"
```

**해결:**
- 중립 옵션 제거
- 강제 선택 (agree or disagree)

**출처:** Likert Scale 연구, Educational Research

---

### 3. 애매한 데이터 제거

**중간값 해석 문제:**
```
People interpret the middle option as:
- "no opinion"
- "don't care"
- "unsure"
- "neutral"
- "it depends"
→ 해석 불가능!
```

**해결:** 중간값 자체 제거

---

### 4. 적용 조건 충족

**중간값 제거가 적합한 경우:**
1. ✅ 응답자가 충분한 정보 가짐 (AI = OK)
2. ✅ 명확한 판단 가능 (정치인 = OK)
3. ✅ Central Tendency Bias 문제 (현재 = OK)
4. ✅ 차별화 목적 (우리 = OK)

**모든 조건 충족 → 중간값 제거 정당함**
