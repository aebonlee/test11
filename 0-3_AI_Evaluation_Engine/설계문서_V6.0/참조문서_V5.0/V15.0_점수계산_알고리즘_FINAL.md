## 🎯 V15.0 핵심 변경사항

### 등급 스케일 변경

**변경 이유**:
- V14.0 실제 데이터 분석 결과, 부정 극단(-8, -7)은 0개
- 긍정 평가가 88%, 부정 평가가 11%로 자연스러운 비대칭 분포
- 현실을 반영하여 긍정 범위 확대, 부정 범위 축소

**V14.0 → V15.0**:
```python
# V14.0 (대칭)
RATING_RANGE = (-8, +8)  # 17단계
부정: -8 ~ -1 (8단계)
중립: 0 (1단계)
긍정: +1 ~ +8 (8단계)

# V15.0 (비대칭)
RATING_RANGE = (-6, +10)  # 17단계
부정: -6 ~ -1 (6단계)
중립: 0 (1단계)
긍정: +1 ~ +10 (10단계)
```

## 📊 V15.0 알고리즘 사양

### 1. 핵심 파라미터

```python
PRIOR = 6.0
COEFFICIENT = 0.5
RATING_RANGE = (-6, +10)  # 17단계
MODEL = "claude-3-5-haiku-20241022"
```

### 2. 점수 계산 공식

```python
# 카테고리 점수 (10개)
category_score = (PRIOR + avg_rating × COEFFICIENT) × 10

# 최종 점수
final_score = sum(category_score_1 ~ category_score_10)
```

### 3. 점수 범위

```python
# 카테고리 점수 범위
최저: (6.0 + (-6) × 0.5) × 10 = 30점
최고: (6.0 + (+10) × 0.5) × 10 = 110점
범위: 30 ~ 110점

# 최종 점수 범위 (이론적)
최저: 30 × 10 = 300점
최고: 110 × 10 = 1,100점
이론적 범위: 300 ~ 1,100점

# V15.0 핵심: 1,000점 상한 적용
final_score = min(raw_score, 1000)
실제 범위: 300 ~ 1,000점
```

**1,000점 상한 적용 이유**:
- 등급 체계 일관성 유지 (V13.0, V14.0 모두 1,000점 기준)
- "완벽한 정치인" (+10 평균)은 이론적 개념
- 1,000점 = 역대 최고 수준의 상징적 의미

## 🎯 V15.0 장점

### 1. 현실 반영
- 실제 데이터 분포 (긍정 88%, 부정 11%)를 스케일에 반영
- 부정 극단(-8, -7)이 거의 없는 현실 수용

- 상위권 정치인 간 차별화 가능

- "극단 사례 포함" 같은 인위적 개입 불필요
- Claude의 자연스러운 평가 패턴 존중

### 4. 스케일 활용도
- 17단계 모두 실제 사용 가능
- V14.0에서 미사용된 -8, -7, +8 제거
