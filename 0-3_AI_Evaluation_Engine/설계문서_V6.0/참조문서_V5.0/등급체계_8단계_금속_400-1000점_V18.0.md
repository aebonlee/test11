## V18.0 알고리즘 적용

### 점수 계산 공식
```python
PRIOR = 6.5
COEFFICIENT = 0.5
RATING_RANGE = -8 ~ +8 (17단계, 대칭)
MAX_FINAL_SCORE = 1000

category_score = (6.5 + avg_rating × 0.5) × 10
raw_final_score = sum(10개 카테고리 점수)
final_score = min(raw_final_score, 1000)  # 1,000점 상한 적용
```

### 점수 범위 계산

```
최소 점수 (모든 rating = -8):
  카테고리 점수 = (6.5 + (-8) × 0.5) × 10 = 25점
  최종 점수 = 25 × 10 = 250점

Prior 기준 (모든 rating = 0):
  카테고리 점수 = (6.5 + 0 × 0.5) × 10 = 65점
  최종 점수 = 65 × 10 = 650점

최대 점수 (모든 rating = +8):
  카테고리 점수 = (6.5 + 8 × 0.5) × 10 = 105점
  이론적 최종 점수 = 105 × 10 = 1,050점
  → 실제 최종 점수 = min(1,050, 1,000) = 1,000점 ✅
```

**이론적 범위**: 250~1,050점
**실제 운영 범위**: 400~1,000점 (등급 체계 기준)

3. **등급 범위 400점 시작**:
